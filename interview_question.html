Interview Exercise


These are the apis available with description:

  1. / :- Home page containing description of apis and interview exercises.
  2. /bets :- Shows all bets
  3. /users :- Shows all users
  4. /affiliates :- Shows all affiliates
  5. /bets/betId :- Data about betId
  6. /affiliates/affiliate_id :- Data about affiliateId
  7. /users/user_id :- Data about userId

Interview Questions:

  1. Find the affiliate with the maximum number of users.
  2. Find the amount won by users coming through the top 3 affiliates (by user_count.)
  3. What is the percentage of users who have won more than 3 bets with low odds - say 25% -
  4. (bonus) Can you make the getting bet by each user concurrent?
  5. (discussion)If you had to build a data pipeline that keeps track of the top 5 users and display every 1 hr how will you do it?
