Interview Exercise


These are the apis available with description:

  1. /users :- Shows all users
  2. /affiliates :- Shows all affiliates
  3. /bets/{betId} :- Data about betId. Sample betId: 111
  4. /affiliates/{affiliate_id} :- Data about affiliateId. Sample affiliateId: 8
  5. /users/{userId} :- Data about userId. Sample userId: 1
  6. /users/{userId}/bets :- All bets by userId

Interview Questions:

  1. Find the affiliate with the maximum number of users.
  2. Find the amount won by users coming through the top 3 affiliates - by user_count.
  3. What is the percentage of users who have won 2 or more bets with low odds - say 25%.

To call an api append the route given to the current url: For example: smarkets.herokuapp.com/users Calls the /users route.
